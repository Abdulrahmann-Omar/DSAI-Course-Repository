## üíª Technical Projects

### Project: Deep Q-Network with Curiosity-Driven Exploration

**Objective**: Improve sample efficiency using intrinsic motivation

**Implementation**:
- DQN with Prioritized Experience Replay
- Intrinsic Curiosity Module (ICM) integration
- Testing on Atari games (Montezuma's Revenge)
- Comprehensive ablation studies

**Results**:
- 2.3x improvement over vanilla DQN
- Significant exploration improvement
- Detailed hyperparameter analysis

[üìÇ GitHub Repository](https://github.com/Abdulrahmann-Omar)

**Technologies**: PyTorch, OpenAI Gym, Weights & Biases, Ray RLlib
